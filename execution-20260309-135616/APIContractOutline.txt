API Contract Outline

1. GET /api/youth-accounts/{youthAccountId}/dashboard
- Returns youth account dashboard details
- Response:
  {
    "youthAccountId": "string",
    "accountBalance": "decimal",
    "spendingLimit": "decimal",
    "currency": "string",
    "recentTransactions": [
      {
        "transactionId": "string",
        "date": "datetime",
        "description": "string",
        "amount": "decimal"
      }
    ]
  }
- Error Handling:
  403 Unauthorized, 404 Not Found, 500 Internal Error

2. POST /api/youth-accounts/{youthAccountId}/fund-transfer
- Allows parent to transfer funds to youth account
- Request:
  {
    "sourceAccountId": "string",
    "transferAmount": "decimal",
    "currency": "string"
  }
- Response:
  {
    "transferId": "string",
    "status": "SUCCESS",
    "updatedBalance": "decimal"
  }
- Error Handling:
  400 Insufficient Funds, 400 Invalid Account, 500 Transfer Service Failure

3. PUT /api/youth-accounts/{youthAccountId}/spending-limit
- Configures weekly spending limit
- Request:
  {
    "weeklyLimit": "decimal"
  }
- Response:
  {
    "youthAccountId": "string",
    "weeklyLimit": "decimal",
    "status": "UPDATED"
  }
- Error Handling:
  400 Invalid Limit, 403 Unauthorized

4. GET /api/youth-accounts/{youthAccountId}/transactions
- Retrieves youth account transaction history
- Query Parameters: startDate, endDate, page, pageSize
- Response:
  {
    "transactions": [
      {
        "transactionId": "string",
        "date": "datetime",
        "merchant": "string",
        "amount": "decimal",
        "transactionType": "debit | credit"
      }
    ],
    "pagination": {
      "page": "number",
      "pageSize": "number",
      "totalRecords": "number"
    }
  }
- Error Handling:
  403 Unauthorized, 404 Not Found, 500 Internal Error

OpenAPI/Swagger specification to be provided as per SCIB-30.
