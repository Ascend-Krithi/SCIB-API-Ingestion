API Contract Outline

1. GET /api/youth-accounts/{youthAccountId}/dashboard
- Description: Retrieve youth account dashboard details
- Request: Path param youthAccountId
- Response:
{
  "youthAccountId": "string",
  "accountBalance": "decimal",
  "spendingLimit": "decimal",
  "currency": "string",
  "recentTransactions": [
    {
      "transactionId": "string",
      "date": "datetime",
      "description": "string",
      "amount": "decimal"
    }
  ]
}
- Errors: 403 Unauthorized, 404 Not Found, 500 Internal

2. POST /api/youth-accounts/{youthAccountId}/fund-transfer
- Description: Transfer funds from parent account to youth account
- Request:
{
  "sourceAccountId": "string",
  "transferAmount": "decimal",
  "currency": "string"
}
- Response:
{
  "transferId": "string",
  "status": "SUCCESS",
  "updatedBalance": "decimal"
}
- Errors: 400 Insufficient funds, 400 Invalid mapping, 500 Transfer failure

3. PUT /api/youth-accounts/{youthAccountId}/spending-limit
- Description: Configure weekly spending limit for youth account
- Request:
{
  "weeklyLimit": "decimal"
}
- Response:
{
  "youthAccountId": "string",
  "weeklyLimit": "decimal",
  "status": "UPDATED"
}
- Errors: 400 Invalid value, 403 Unauthorized

4. GET /api/youth-accounts/{youthAccountId}/transactions
- Description: Retrieve youth account transaction history
- Request: Query params startDate, endDate, page, pageSize
- Response:
{
  "transactions": [
    {
      "transactionId": "string",
      "date": "datetime",
      "merchant": "string",
      "amount": "decimal",
      "transactionType": "debit | credit"
    }
  ],
  "pagination": {
    "page": "number",
    "pageSize": "number",
    "totalRecords": "number"
  }
}
- Errors: 403 Unauthorized, 404 Not Found

Common Error Handling:
- All endpoints return standardized error object:
{
  "errorCode": "string",
  "errorMessage": "string"
}

Security:
- OAuth token validation required
- Parent ownership validation enforced
- RBAC on all endpoints
