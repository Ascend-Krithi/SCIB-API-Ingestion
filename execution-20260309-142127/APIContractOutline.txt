API Contract Outline

1. GET /api/youth-accounts/{youthAccountId}/dashboard
Request: Path param youthAccountId
Response:
{
  "youthAccountId": "string",
  "accountBalance": "decimal",
  "spendingLimit": "decimal",
  "currency": "string",
  "recentTransactions": [
    {"transactionId": "string", "date": "datetime", "description": "string", "amount": "decimal"}
  ]
}
Error Handling:
- 403 Forbidden (unauthorized parent)
- 404 Not Found (invalid youth account)

2. POST /api/youth-accounts/{youthAccountId}/fund-transfer
Request:
{
  "sourceAccountId": "string",
  "transferAmount": "decimal",
  "currency": "string"
}
Response:
{
  "transferId": "string",
  "status": "SUCCESS",
  "updatedBalance": "decimal"
}
Error Handling:
- 400 Bad Request (insufficient funds, invalid mapping)
- 500 Internal Server Error (transfer service failure)

3. PUT /api/youth-accounts/{youthAccountId}/spending-limit
Request:
{
  "weeklyLimit": "decimal"
}
Response:
{
  "youthAccountId": "string",
  "weeklyLimit": "decimal",
  "status": "UPDATED"
}
Error Handling:
- 400 Bad Request (invalid value)

4. GET /api/youth-accounts/{youthAccountId}/transactions?startDate=&endDate=&page=&pageSize=
Request: Path param youthAccountId, query params
Response:
{
  "transactions": [
    {"transactionId": "string", "date": "datetime", "merchant": "string", "amount": "decimal", "transactionType": "debit | credit"}
  ],
  "pagination": {"page": "number", "pageSize": "number", "totalRecords": "number"}
}
Error Handling:
- 403 Forbidden (unauthorized parent)
- 404 Not Found (invalid youth account)

Authentication:
- OAuth token required for all endpoints

General:
- All endpoints return application/json
- Validation errors return HTTP 400
- Unauthorized access returns HTTP 403
- All actions logged for audit
